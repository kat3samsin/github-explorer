# github-explorer
A GitHub repository explorer created using [create-react-app](https://facebook.github.io/create-react-app/).

### Tech Stack
- react
- redux
- redux-thunk

### Testing
- jest
- enzyme

# installation
1. git clone https://github.com/kat3samsin/github-explorer.git
2. cd github-explorer
3. npm install
4. npm start
